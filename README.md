## 🛠 使い方

### ● 1. AI に渡す
AI に以下のように伝えるだけで、  
あなたの価値観を正確に理解させられます。

### ● 2. 自己理解のために読み返す
- vim で開く  
- GitHub で読む  
- スマホで眺める

あなたの価値観や対話スタイルが進化したら、  
このファイルも更新していくことで  
**“人格アーキテクチャのバージョン管理”** ができます。
